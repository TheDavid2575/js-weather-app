App Information


How to use: Write in the name of a city anywhere around the world and hit search! Switch between farnheight and celcius with the button on the top right.

Tools used: HTML, CSS, JavaScript, OpenWeatherMap API & a bit of bootstrap.

For geeks: This weather app was built with plane old JavaScript. Obtaining the information from OpenWeatherMap.org's weather API. Data is obtained using the old-fashioned technique; XMLHttpRequest objects.

Concepts applied: AJAX using XMLHttpRequest objects, JSON manipulation, obtaining information from an API, DOM Manipulation.
